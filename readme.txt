== Description ==

Pagamento QR Code Plugin Ame Digital, desenvolvido com as melhores práticas de desenvolvimento. Proporcionando mais agilidade, segurança e conversão de vendas.

Veja o [homolog](https://amedigital.apikihomolog.com/) com o tema Storefront.

= Requirements =

- PHP version 7.1 ou maior.
- WooCommerce version 4.0.x ou maior.

= Payments methods =

- Obrigatório APP Ame Digital.

== Installation ==

1. Faça upload deste plugin em seu WordPress, e ative-o;
2. Entre no menu lateral "WooCommerce > Configurações > Pagamentos > Ame";
3. Adicione o Access Token ( entre em contato com a AME e solicite ).
4. Adicione Título, Descrição, Título do checkout, Status do pedido e habilite o metódo, depois salve alterações.
